# Log Parser (v0.1 — 初始化)

## 📌 專案目標
使用 C++17 + STL 在 Linux 上開發一個簡單的 Log Parser 工具。
未來功能包含：
- 讀取 log file
- 印出每一行內容
- tokenize 每一行
- keyword 統計
- CLI 參數 (--key, --file)
